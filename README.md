# Student Performance Analysis

## Objective
Analyze factors affecting student performance using a dataset from Kaggle.

## Dataset
Students Performance in Exams dataset.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Key Insights

1. Female students perform better in reading and writing.
2. Male students score slightly higher in math.
3. Students who completed the test preparation course score higher.
4. Math has the lowest average score among the three subjects.
